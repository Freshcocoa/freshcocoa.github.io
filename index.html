History

All project events can be easily tracked from the history tab, giving you a great overview of all project developments.

Merge Requests

Compose, view, update, comment and approve merge requests. Dive in the detail and see exactly what's changing with the diff view.

Issues

Easily create, edit and close/reopen issues. Issues can also be commented on and assigned to team members and milestones.

Project Files

Project files and folders can be easily browsed from a specific branch or commit.

Self Signed Certificate Support

If you don't use a standard Certificate Authority, don't worry! Trident will prompt you to trust a certificate based on the SHA1 fingerprint.

Certificate Pinning

Trident supports Certificate Pinning, so you can be sure you're using the correct Certificate.
